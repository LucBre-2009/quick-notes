# Minimal Notes

A minimal note-taking website for GitHub Pages.

## Features

- Create notes
- Delete notes
- Automatic browser storage
- Automatic deletion after 24 hours
- JSON export & import
- German / English support
- Responsive design

## Files

- `index.html`
- `style.css`
- `script.js`

## Usage (local)

1. Download the files
2. Put them into one folder
3. Open `index.html`

## Usage (on github pages with easy access)

1. Open the [Website](https://lucbre-2009.github.io/quick-notes/)
2. and use it :)


## Notice

All data is stored locally in the browser only.
